## 📖 Introduction
**WynnPerf** is an optimized modpack specifically tailored for adventurers in the kingdom of **WynnCraft**. 

Built on the foundation of **Valkaw**, we have implemented a "de-bloating" process—removing unnecessary mods to achieve the highest possible frame rates—while integrating advanced RPG support features for a smoother and more intuitive gameplay experience.

## 📥 Installation Guide

### 🟢 Modrinth App, MultiMC, Prims Launcher
1. Open the launcher that supports the Modrinth modpack format.
2. Search for **WynnPerf**.
3. Choose your preferred edition, either Sodium or VulkanMod, and the Minecraft version you want to play.
4. Click **Install** and hit play! 

### 🛠️ Manual Installation

1. Install Fabric Loader from [fabricmc.net](https://fabricmc.net/use/installer/).
2. Locate your Game Folder: open your launcher and find the "Game Directory" or "Open Folder" button. Standard path: `%appdata%\.minecraft` (Windows) or `~/Library/Application Support/minecraft` (macOS).
3. Extract Modpack Files: extract the ZIP file u download from GitHub and copy the following folders and files from the zip into your `.minecraft` directory: **mods - config - resourcepack - options.txt**
4. Allocate RAM: I recommend for playing WynnCraft is allocate 4GB of RAM
* Go to **Settings** or **Edit Profile** in your launcher.
* Find **JVM Arguments** or **Memory Allocation**.
* Change `-Xmx1G` to `-Xmx4G` or `-Xmx6G`.
